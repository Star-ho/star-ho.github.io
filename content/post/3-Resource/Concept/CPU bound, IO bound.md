## Cpu Bound

* CPU가 빨라지면 작업이 빨라지는 작업
* CPU를 사용하는것이 주요한 작업일 경우

> π값 계산

## I/O bound

* I/O 작업이 많은것
* ex) 네트워크, 파일읽기

> 거대한 파일에서 특정 데이터 찾기
